# SyntaxWear

SyntaxWear é um projeto de e-commerce de página única para uma loja de tênis e sneakers online. O projeto foi desenvolvido utilizando apenas HTML e CSS, com foco em um design moderno e responsivo.

## Visão Geral

O site apresenta um layout limpo e atraente, com as seguintes seções:

*   **Cabeçalho:** Navegação fixa com links para as categorias de produtos, informações da loja e conta do usuário.
*   **Herói:** Uma seção de destaque com uma imagem de fundo e chamadas para ação.
*   **Categorias de Produtos:** Cards que representam as diferentes categorias de tênis disponíveis.
*   **Grid de Produtos:** Um layout de grid para exibir os produtos em destaque.
*   **Rodapé:** Informações de contato, links de navegação e um formulário de inscrição para newsletter.

## Tecnologias Utilizadas

*   **HTML5:** Para a estrutura semântica do site.
*   **CSS3:** Para estilização, layout e responsividade.
    *   **Flexbox:** Utilizado para alinhar e distribuir os elementos de forma flexível.
    *   **Grid Layout:** Para criar o layout complexo da grade de produtos.
    *   **Media Queries:** Para garantir que o site seja totalmente responsivo em diferentes dispositivos.

## Como Abrir o Site

Para visualizar o site, basta abrir o arquivo `index.html` em seu navegador de preferência.

1.  Clone o repositório:
    ```bash
    git clone https://github.com/alanborgesdev/ecommerce-syntaxwear.git
    ```
2.  Navegue até o diretório do projeto:
    ```bash
    cd ecommerce-syntaxwear
    ```
3.  Abra o arquivo `index.html` no navegador.
